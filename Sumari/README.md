# DuelCut context-n-goal folder

This folder is meant to be pasted into the project root and used as context for TRAE SOLO.

Recommended usage:

1. Read `00_PROJECT_CONTEXT.md` first.
2. Each team member reads their own file:
   - Person 1: `01_PERSON_1_UI_PRODUCT_CONTEXT.md`
   - Person 2: `02_PERSON_2_LOGIC_PROMPT_CONTEXT.md`
   - Person 3: `03_PERSON_3_REPLAY_VIDEO_CONTEXT.md`
3. Everyone follows `04_PHASES_AND_TESTCASES.md`.
4. Use prompts from `05_MASTER_TRAE_PROMPTS.md` inside TRAE.

Main rule:

Do not over-scope. Build a reliable vertical slice.

sample battle log -> 5 storyboard scenes -> PixVerse-ready prompts -> 30s+ replay timeline -> timestamp comments -> share/export.
